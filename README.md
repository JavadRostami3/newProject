# Teacher Dashboard (Vite + React)

A teacher dashboard UI built with Vite + React and styled with Tailwind CSS.

## Tech

- React
- Vite
- Tailwind CSS
- Recharts
- dayjs + jalaliday (Jalali date support)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
